# Example Epic Initial Idea

> **Template-only example.** This file and its whole `example-epic/` directory
> show the epic planning layout. Do not copy them into a target repository.

This note is written by a human. It can be rough, but it is the source of truth
for the epic's starting intent. Agents may clarify it with the user. They must
not silently replace its meaning.

## Initial idea

Let workspace admins request large report exports without keeping the browser
open. Show progress, keep completed files for a short time, and let admins
cancel work that has not finished.

## Early limits

- Only workspace admins can request or download an export.
- Export work must not slow down normal report views.
- The exact file lifetime still needs a user decision.
